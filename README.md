# Low Input Lag Windows Setup

This repository contains a collection of configuration ideas and useful information that may help gamers reduce input latency on Windows systems.

Competitive gamers often adjust system settings to minimize input delay between keyboard, mouse, and the game engine.

Lower input latency can improve responsiveness, aiming precision, and overall gameplay performance.

Here you can find several example configuration ideas related to system performance, graphics settings, and device optimization.

Before applying any system changes, review the settings carefully and adjust them based on your hardware configuration.
